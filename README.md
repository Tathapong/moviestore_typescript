# Todolist

### How to Run

Clone project

```
git clone https://github.com/Tathapong/moviestore_typescript
```

go to project

```
cd todolist_typescript
```

install npm

```
npm install
```

Insert your API_KEY

```
create .env.local in  /src
define parameter API_KEY (api key of TMDB)
```

start project

```
npm run dev
```
